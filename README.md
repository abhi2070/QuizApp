#QuizApp
Key Features: 
RESTful APIs for quiz and question management 
JWT-based authentication with role-based authorization 
Spring Security integration for endpoint protection Centralized 
global exception handling using @ControllerAdvice 
DTO-based request/response mapping for clean API contracts 
Validation using Bean Validation (JSR-380) 
Layered architecture: Controller → Service → Repository 

#Tech Stack 
Backend : Java, Spring Boot 
Security : Spring Security, 
JWT Database : MySQL 
Persistence : Spring Data JPA 
Build Tool : Maven 
Tools : IntelliJ IDEA, Postman
